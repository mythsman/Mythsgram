# Mythsgram

## Introduction
A simple project for [Instagram](https://www.instagram.com) , using SpringBoot structure.

## Function
* Login and logout.
* Profile update.
* Sensitive word filter.
* Picture upload , size limit ,type limit.
* Comment and star record with redis.
* Responsive layout.
* Anti script injection.
